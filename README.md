# wormhole
A wormhole which connects development environments

## Prerequisites
- Bash
- Git
- Neovim, Luajit
- Alacritty
- Zellij

## Workflows
- Lua
- Clang
- Rust
- Python
- Go
- Latex

## Deployment
```sh
sh deploy.sh
```
